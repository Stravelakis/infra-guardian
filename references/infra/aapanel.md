# aaPanel

## In plain English
A free web-based control panel for managing a Linux server through a
point-and-click dashboard instead of typing commands. Handles websites,
databases, SSL certificates, and background services for you.

## Where I use it
- Self-hosted servers (my own VPS / dedicated boxes)
- Managing WordPress sites and their databases

## Good to know
- Access is via a browser on a custom port (default 7800/8888).
- Always secure the login with a strong password + firewall rule.
- Keep it updated — panels are a common attack target.
