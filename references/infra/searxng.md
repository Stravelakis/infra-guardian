# SearXNG

## In plain English
A self-hosted, privacy-respecting metasearch engine. It queries many
search engines at once and blends the results, without tracking you or
storing your searches.

## Where I use it
- Private search that doesn't profile me
- A search backend for AI tools/agents that need web results

## Good to know
- Runs well as a Docker container.
- Configuration lives in a YAML settings file.
- Can be paired with a proxy so agents call it via a local API.
