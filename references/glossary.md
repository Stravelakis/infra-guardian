# Glossary — Plain-English term bank

This file grows over time. Every new acronym or dev term the skill uses
gets one plain-English line here, so the vocabulary compounds.

| Term | In plain English |
|------|------------------|
| systemd | The part of Linux that starts and supervises background programs. |
| timer | A schedule that runs a task automatically (like an alarm clock). |
| YAML | A simple text format for settings files. Indentation matters. |
| .env | A hidden file holding secret keys. Never shared or uploaded. |
| checksum | A fingerprint of a file used to prove it wasn't corrupted. |
| dry-run | A preview that shows what would happen without changing anything. |
| drift | When the real setup no longer matches what was expected. |
| repo | A project folder tracked by Git (version history). |
| token | A secret password-like string that lets software log in to a service. |
| daemon | A program that runs quietly in the background. |
| API key | A password-like code that lets one app use another app's service. |
| environment variable | A setting stored outside your files that programs can read at run time. |
| DNS | The internet's phone book — turns a domain name into a server address. |
| A record | A DNS entry that points a domain to a server's numeric address. |
| CNAME | A DNS entry that makes one name an alias for another name. |
| MX record | A DNS entry that says where a domain's email should be delivered. |
| TXT record | A DNS entry holding text, often used to verify ownership. |
| TTL | How long a DNS answer is remembered before it's looked up fresh. |
| nameserver | The service that officially holds a domain's DNS records. |
| zone | The full set of DNS records for one domain. |
| propagation | The wait for a DNS change to spread across the internet. |
| registrar | The company you buy and renew a domain name from. |
| commit | A saved snapshot of your project in Git, with a note about what changed. |
| push | Uploading your saved snapshots to the online copy of the project. |
| gitignore | A list telling Git which files to never upload (e.g. secrets). |
| placeholder | A stand-in value (like <paste-key>) you replace with a real one later. |
| CDN | A network of servers worldwide that delivers your site faster. |
| SSL / TLS | The tech behind the padlock — encrypts traffic so it's private (https). |
| snapshot | A saved copy of something's exact state at one moment, for restoring later. |
| rotate (a key) | Replacing a key/token with a new one and disabling the old one, so a leaked or old key stops working. |
