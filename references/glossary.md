# Glossary — Plain-English term bank

This file grows over time. Every new acronym or dev term the skill uses
gets one plain-English line here, so the vocabulary compounds.

| Term | In plain English |
|------|------------------|
| systemd | The part of Linux that starts and supervises background programs. |
| timer | A schedule that runs a task automatically (like an alarm clock). |
| YAML | A simple text format for settings files. Indentation matters. |
| .env | A hidden file holding secret keys. Never shared or uploaded. |
| checksum | A fingerprint of a file used to prove it wasn't corrupted. |
| dry-run | A preview that shows what would happen without changing anything. |
| drift | When the real setup no longer matches what was expected. |
| repo | A project folder tracked by Git (version history). |
| token | A secret password-like string that lets software log in to a service. |
| daemon | A program that runs quietly in the background. |
